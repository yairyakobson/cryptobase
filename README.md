# Cryptobase - Your Main Cryptocurrency Base

<h4>Server: NodeJS, express, mongoose</h4>
<h4>Client: React, TailwindCSS</h4>
<h4>Database: MongoDB</h4>
<h4>Media Storage: AWS S3</h4>
